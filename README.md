PickMe is a full-featured, backend-powered voting system that enables organizations to create and manage voting sessions efficiently. 
It offers real-time vote tracking, secure voter access, and an admin dashboard for session control — all in one streamlined platform.
**Features**
* For Organizations
  - Create and manage multiple voting sessions
  - Add, edit, or remove contestants
  - Generate unique session links for voters
  - View real-time results and analytics
* For Voters
  - Join voting sessions through organization-specific links
  - Cast votes securely and anonymously
  - See live vote counts (if enabled by admin)

**Backend Highlights**
  * RESTful API with secure authentication
  * MongoDB for efficient data storage
  * WebSocket-based real-time vote tracking
  * Built for scalability and modularity

**Tech Stack**
  Backend: Node.js, Express.js
  Database: Prisma + Supabase
  Authentication: JWT
  Real-Time Updates: Socket.io
